Documentation & Philosophy

What is RoastMe?

Originally designed just for code, this AI tool has evolved into a universal, ruthless critic. Paste in your code, an email draft, a bad startup idea, or your dating app bio. Our AI will provide hilariously savage feedback—and then actually help you fix it.

How it Works

  1. Paste your text: Drop your code, email, or essay into the "Raw Meat" section.
  2. Ignite the Grill: The app streams your input to a frontier AI model via OpenRouter.
  3. Get Roasted: The AI identifies the biggest flaws, roasts you mercilessly, provides the correct rewritten version, and gives you a professional tip.

Configuration (For Developers)

If you downloaded this source code, you need to set your own API key.

// Open js/app.js and update the CONFIG object:
const CONFIG = {
  apiKey: "sk-or-...", // Your OpenRouter API Key
  model: "google/gemma-7b-it" // Or any model you prefer
};

The AI Persona

We specifically engineered the AI prompt to avoid generic "AI speak" (like "As an AI language model..."). Instead, it responds directly in a strict format:

  • 🔥 The Roast: A punchy critique of your code.
  • ✅ The Fix: A syntax-highlighted code block showing the right way.
  • 💡 Pro Tip: A sarcastic but genuinely useful best practice.
  • 💀 Score: A rating out of 100 on the garbage-fire scale.
